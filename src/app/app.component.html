NgPaint
Informations
MenuDirective
validateText: String Texte de validation des boutons d'effets
cancelText: String Texte du bouton de retour arrière
importText: String Texte du bouton d'import d'image
color: String Couleur du menu; red(défaut), blue, green
menu: Array<MenuElementDirective> Tableau représentant le menu
MenuElementDirective
effect: String Nom de l'effet à appliquer
title: String Titre de l'effet dans le menu
redirect: String Menu vers lequel on se redirige en cliquant
submenu: String Sous-menu dans lequel se situe le bouton
*Un élément ne peut pas avoir un effet et une redirection
DataBlob
@Output() récupérant les données de l'image sous la forme d'un Blob lors de l'appel du téléchargement

Nom Effet
reset Remet l'image telle qu'importée
changePicture Permet de changer l'image utilisée
download Émet l'outPut dataBlob avec le blob de l'image png comme paramètre
basicDrawForm Menu de sélection des formes de dessin(pinceau, carré, rond, flèche, étoile, texte)

sizeSlider Slider de sélection de la taille pour les formes de dessin

contrast Permet d'ajuster le contraste de l'image
saturation Permet d'ajuster la saturation de l'image
inver Inverse les couleurs de l'image
grayscale Transforme les couleurs de l'image en teinte de gris
sepia Applique un filtre jaune/brun sur l'image
candy Applique un filtre rosé sur l'image
rainbow Applique un filtre arc-en-ciel sur l'image
gaussian Applique un léger flou gaussien sur l'image
sharpen Applique un filtre affinant les contours de l'image (en progrès)
vignette Rajoute une vignette noire sur les bords de l'image
warmer Réchauffe les couleurs de l'image
colder Refroidit les couleurs de l'image
emphasing Transforme l'image en changeant une des couleurs primaires à la valeur sélectionnée
splash Transforme les couleurs de l'image en teinte de gris si elles sont trop éloignées de la couleur sélectionnée
